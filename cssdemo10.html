10. CSS Box Model

Every HTML element is considered a "box". The box model describes how these boxes are rendered. It consists of: **Content**, **Padding**, **Border**, and **Margin**.

Margin (transparent)
Border (dashed blue)
Padding (light blue)
Content (blue) This is the actual content area.

**Content:** The actual content of the box, where text and images appear.
**Padding:** Clears an area around the content. It is transparent.
**Border:** A border that goes around the padding and content.
**Margin:** Clears an area outside the border. It is transparent.
